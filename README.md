# RoomAIgine

## Core Features:

- Room Restyling: AI-powered room restyling using Puter.js (puter.ai.image.generate) based on user-uploaded photo, providing different design styles (minimalist, luxury, cozy, industrial). The LLM acts as a tool to interpret and implement design principles, influencing image composition, color palettes, and the placement of furniture elements.
- Style Grid Display: Display the different room style variations in a grid layout for easy comparison.
- Downloadable Images: Functionality to download the generated styled images for later use.
- Shareable Designs: Provide options to share the styled images on social media platforms.
- Furniture Suggestions: Suggest matching furniture items that complement the selected room style, and provide direct purchase links.
- AR Furniture Preview: Enable Augmented Reality (AR) mode to preview selected furniture items in the user's actual room environment.

## Style Guidelines:

- Primary color: Deep Indigo (#663399) to evoke creativity and sophistication.
- Background color: Light gray (#F0F0F0) for a neutral backdrop that highlights the restyled images.
- Accent color: Muted violet (#800080) for highlighting interactive elements.
- Body and headline font: 'Inter', a sans-serif for a clean and modern aesthetic.
- Responsive grid layout that adapts to different screen sizes, optimizing user experience on desktop and mobile devices.
- Smooth transitions and loading animations to enhance user engagement while the AI generates and restyles the images.
- Use simple, minimalist icons to represent different design styles and functionalities, in a style that avoids being cartoonish, and avoids free-use iconography styles.
